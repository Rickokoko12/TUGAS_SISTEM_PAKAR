# SISTEM PAKAR DIAGNOSA PENYAKIT MATA PADA ANAK

Metode: Forward Chaining  
Teknologi: PHP & MySQL  

## Deskripsi
Aplikasi sistem pakar untuk membantu mendiagnosa penyakit mata pada anak
berdasarkan gejala yang dipilih pengguna.

## Cara Menjalankan
1. Simpan folder project di `htdocs`
2. Import database ke phpMyAdmin
3. Jalankan melalui browser: `http://localhost/`

## Catatan
Project ini dibuat untuk memenuhi tugas mata kuliah Sistem Pakar.
